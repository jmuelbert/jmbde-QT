6.3. Command line options¶

jmbde-QT is designed to be as user-friendly as possible. As such, its command line options are only useful for development or debugging, as they make these tasks easier. Command line options are case-sensitive.

New in version 0.4.0:

  • --help option

Changed in version 0.4.0:

  • --help option prints the version and an help messages

Description

–help
