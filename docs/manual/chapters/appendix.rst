.. _appendix-command-line-options:

Command line options
====================

jmbde-QT is designed to be as user-friendly as possible. As such, its command line
options are only useful for development or debugging, as they make these tasks
easier. Command line options are case-sensitive.

.. versionadded:: 0.4.0

   * ``--help`` option

.. versionchanged:: 0.4.0

   * ``--help`` option prints the version and an help messages

======================  =================================================
Option                  Description
======================  =================================================
--help                  Print the program version and an help message
======================  =================================================
