Getting Started
***************

Opening jmbde
=============
|logo| Once you've `installed jmbde <https://github.com/jmuelbert/jmbde-QT/releases>`_, start by opening
jmbde add data...

**Windows**
  Double-click the jmbde icon on the Desktop. Alternatively, browse your Windows
  start menu and click the jmbde icon, or perform a search for
  :file:`jmbde.exe`.

**Mac OSX**
  Double-click the jmbde icon in the :file:`Applications` folder. Alternatively,
  drag the jmbde icon to the dock and double-click it there or search for
  :command:`jmbde` in Spotlight.

**GNU/Linux**
  Click the jmbde icon in the applications menu or launcher of your desktop
  environment or perform a search for :file:`jmbde`. Alternatively type
  :command:`jmbde` into the terminal, then hit :kbd:`Return`.
