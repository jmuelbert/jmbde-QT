JMBDE

Ein kleines Werkzeug zur Betriebsdatenerfassung

Mit JMBDE können viele Daten erfasst werden. Das sind Mitarbeiter, Computer einschließlich der Server, Drucker, Telefone und Faxgeräte.

Diese erfassten Daten können dann miteinander verknüpft werden. Die Daten werden in einer Datenbank erfasst. Diese SQL-Datenbank ist schnell.
