jmbde 0.5.4 User Manual¶

jmbde is a free <https://www.gnu.org/philosophy/free-sw.html> software for Windows, macOS and Linux. This program is easy to understand and easy to use. With this software you can record and manage employees, hardware and accesses in a company.
